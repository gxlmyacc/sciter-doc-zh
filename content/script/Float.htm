Float(浮点)对象

浮点值是这个类的"对象"。

常量

MIN
- float, 浮点类型的最小值。
MAX
- float, 浮点类型的最大值。

方法

toFloat

( ) : float

简单的返回浮点值。

toInteger

( ) : int

将浮点值转换成整数。

toString
( )

返回浮点数的字符串形式。

toHtmlString

() : string

返回转义了html规则的字符串。是toString()方法的别名。

toUrlString

() : string

返回转义了url规则的字符串。是toString()方法的别名。

valueOf
( ) : float

简单的返回浮点值本身。

min

( ... ) : float

静态方法 - 返回参数中的最小值。如Float.min(1.0, 2.0, 3.0)将会返回1.0。

max

( ... ) : float

静态方法 - 返回参数中的最大值。如Float.min(1.0, 2.0, 3.0)将会返回3.0。

isNaN
( ) : true|false

如果该浮点值无法表示为数字，则返回true。

isFinite
( ) : true|false

如果该浮点数是一个无穷大数字，则返回true。

morph

( from, to, ratio ) : float

使用后面的公式计算平均值: (to - from) * ratio + from;
