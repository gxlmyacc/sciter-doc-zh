Math(算法)对象

算法对象没有构造函数，因此你无法创建一个它的实例。可以将Math作为一个名称空间——公共静态方法和常量的集合。

属性(常量)

E
- float, e - 欧拉常数，自然对数的底。
LN2
- float, 2的自然对数的值。
LN10
- float, 10的自然对数的值。
LOG2E
- float, e 常量以2为底的对数的值。
LOG10E
- float, e 常量以10为底的对数的值。
PI
- float, Pi 常量 - 圆周率 - 圆的周长与直径的比值。
SQRT1_2
- float, 0.5的平方根的值。
SQRT2
- float, 2.0的平方根的值。

方法

abs

( integer | float )

返回该函数浮点或整数参数的绝对值。

sin
cos

( integer | float ) returns: float

返回参数的正弦或余弦函数值。

tan
( integer | float ) returns: float

返回参数的正切函数值。

atan
( integer | float, [integer | float] ) returns: float

返回参数的反正切函数值。内部调用的C运行时的一个参数的atan()或两个参数的atan2()函数。

asin
acos
( integer | float ) returns: float

返回参数的反正弦或反余弦值。

sqrt

( integer | float ) returns: float

返回参数的平方根值。
ceil

( float ) returns: integer

返回大于或等于该参数的最小整数值。

floor

( float ) returns: integer

返回的小于或等于参数的最大整数值。

round
( float ) returns: float

返回参数的四舍五入值。Math.round(3.2) -> 3.0 以及 Math.round(3.7) -> 4.0

exp

( integer | float ) returns: float

如果成功，返回参数的指数值。如果溢出，则返回INF (无穷大)。如果下溢，则返回0.0。若要测试数值是否无穷大请使用Float.isFinite(value)方法。

log
log2
log10
( integer | float ) returns: float

如果成功，则返回参数的对数值。如果参数时负数，则返回不确定值(NaN)。如果参数是0，则返回INF(无穷大)。若要测试数值是否无穷大请使用Float.isFinite(value)方法。

pow

( number, exponent ) returns: float | integer

计算number的exponent的指数幂: result = numberexponent

random
( ) returns: float

返回0.0 ... 1.0之间的一个随机浮点数。
