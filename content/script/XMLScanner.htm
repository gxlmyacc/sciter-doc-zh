XMLScanner对象

XMLScanner - XML/HTML tokenzier。也被称为XML解析器。

常量

token()方法的返回值。

ERROR
- 在输入流中的标记错误。
EOF
- 到达流的末尾。
HEAD
- 解析到标签元素头开始:
<tag ...
  ^-- 该常量产生自这里。
HEAD_END
- 解析到非空标签元素头结束:
<tag ... >
       ^-- 该常量产生自这里。
EMPTY_HEAD_END
- 解析到空标签元素头结束:
<tag ... />
        ^-- 该常量产生自这里。
TAIL
- 解析到空标签元素尾开始:
</tag>
 ^-- 该常量产生自这里。
ATTR
- 解析到属性:
<tag attr="value" >
                ^-- 该常量产生自这里。

属性可以有或没有(html样式)值。scanner.attribute是属性的名，scanner.value是属性的值。

TEXT
-解析到文本.

scanner.value包含该文本。

CDATA
- 解析到cdata:
<![CDATA[ ...value... ]]>
                     ^-- 该常量产生自这里。

scanner.value包含cdata的文本。

PI
- 解析到处理指令:
<? ...value... ?>
              ^-- happens here

scanner.value 包含指令的文本。

DOCTYPE
- 解析到doctype声明: 
<!DOCTYPE ...value... >
                      ^-- 该常量产生自这里。

scanner.value 包含doctype的声明文本: <!DOCTYPE 后面的和 '>' 前面的字符串。

属性

value
- string, 属性的值、文本、cdata或指令文本等。
attribute
- string, 属性名。当token == XMLScanner.ATTR时有效。
tag
- string, 标签名。当token为XMLScanner.HEAD、XMLScanner.TAIL或XMLScanner.HEAD_END时有效。

方法

token

(  ) returns: int

返回一个上面的常量。请使用它们的全格式，如XMLScanner.HEAD、XMLScanner.TAIL等。
